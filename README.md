## Contributing

We welcome contributions! Here's how you can help:

### Adding a New Conference

1. **Fork this repository**
2. **Add your conference** using the template at the top
3. **Place it in the appropriate section** (chronologically by date within section)
4. **Submit a Pull Request** with a clear description

### Conference Criteria

To be included, conferences should:

- Focus on computer vision, machine learning, AI, or related fields
- Have a **fixed annual location** (not rotating through multiple cities per month)
- Be recurring events with peer-reviewed submissions
- Have a public website

### What to Include

- Official website URL
- Exact dates (or "TBA")
- City and country
- Brief style/audience description
- CFP dates and link (if applicable)

### Updating Information

Found outdated info? Submit a PR with:

- Updated dates
- New CFP deadlines
- Corrected links

### Reporting Issues

Open an Issue with links to official sources.
